# ✈️ Flight Management System

## 📖 Abstract

The Flight Management System is a Java-based booking solution for flight tickets. It consolidates data provided by different airline carriers and provides users with flight details and rates in real time.

Travellers may want to make changes to their bookings. The application allows them to book, cancel, view, and update their bookings with ease. In addition, it simplifies the management of bookings for administrators. All bookings, flights, schedules, and routes can be viewed, added, and modified through a single application.

---

## 🎯 Scope

### ✅ In Scope

The system provides the following functionalities.

There are two categories of users who can access the system:

- **Customer**
- **Administrator**

Each category has its own set of privileges.

#### Customer

- Create a user account.
- Log in to the application.
- Check available flights.
- Make a booking.
- View bookings.
- Cancel a booking.
- Modify a booking.

#### Administrator

- Log in to the application.
- Add flight, schedule, and route details.
- View flight, schedule, and route details.
- Cancel or modify flight, schedule, and route details.

---

## 🚫 Out of Scope

The following functionalities are not covered by the application:

1. Boarding pass generation and seating plan management.
2. Third-party integrations such as email and SMS services.
3. Payment processing.
